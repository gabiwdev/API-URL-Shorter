# Encurtador de URL

Um projeto full-stack simples, com foco em back-end para aprendizado de APIs e comunicação entre front-end e back-end.

# Como usar

Ao baixar os arquivos e ter "Python 3" instalado no computador, basta acessar a pasta através de um terminal e executar
```bash
python main.py
```
Assim um localhost será aberto, a página exibirá um campo para digitar uma URL.
Ao digitar a url e clicar em "encurtar" será retornada a URL encurtada pronta pra uso.

# Conceitos aprendidos

- Aprofundado o conhecimento geral sobre APIs
- Melhor entendimento da relação entre Front-end e Back-end
- Conhecimento dos métodos HTTP: GET e POST
